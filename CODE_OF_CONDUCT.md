# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

- Be respectful and inclusive.
- Assume good intent, give constructive feedback.
- No harassment, discrimination, or personal attacks.

## Reporting

Report unacceptable behavior to the maintainers via [GitHub Issues](https://github.com/mcbuddy/testivai-oss/issues) or by emailing the project maintainers.

## Enforcement

Maintainers may warn, edit, or remove contributions and contributors who violate this code of conduct.
